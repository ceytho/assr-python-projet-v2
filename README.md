# Projets module Python ASSR

Ce dossier contient les sujets de projet Python ASSR

- Le projet se fait en **groupes**
- Chaque groupe doit choisir **un sujet distinct**

## Évaluation

La notation du module repose sur deux volets :

- un **livrable** à rendre le **dernier jour du projet**
- une **présentation orale**

## Présentation orale

La présentation orale est obligatoire pour tous les groupes.  
Elle doit inclure une démonstration du projet et un support de présentation.

Le support doit couvrir au minimum :

- la composition du groupe et la répartition du travail
- l'architecture globale de la solution
- les choix techniques principaux
- les difficultés rencontrées et les solutions apportées

## Livrable attendu

#### Ce qu'il y a dedans

- le code source Python
- le fichier de configuration
- la documentation d'utilisation
- la liste des dépendances (`requirements.txt` si nécessaire)
- le support de présentation orale

**À envoyer à cette adresse avec la présentation : [assrdepotpython@outlook.fr](mailto:assrdepotpython@outlook.fr)**

#### Qualité de code attendue

- Le code doit être découpé en **fonctions** avec des responsabilités claires
- Les fonctions doivent être commentées avec des docstrings **(objectif fonction, params, valeur de retour, exceptions éventuelles)**
- Le code doit être commenté (attention, pas à outrance, pas chaque ligne, choisissez soigneusement)
- Vous devrez gérer tous les edge-cases avec une **gestion d'erreur/exceptions**
- Les choix d'architecture doivent être cohérents et justifiables à l'oral

## Concernant l'intelligence artificielle

#### Oui c'est l'avenir mais ça ne remplacera pas vos cerveaux

L'usage des LLM est toléré car faisant partie de la réalité du terrain aujourd'hui dans notre domaine, néanmoins vous devrez être capable d'expliquer à l'oral chaque ligne de code et chaque choix architectural si demandé

La note sera fortement impactée si le livrable est parfait et qu'à la présentation orale un manque de connaissances se fait sentir (pas de flicage, mais je préfère souligner la méthode d'évaluation)

En revanche profitez de ce projet pour apprendre de nouvelles choses sans tricher, l'IA doit être un assistant, et ne doit pas remplacer votre cerveau en tant que professionnels de l'IT :)
